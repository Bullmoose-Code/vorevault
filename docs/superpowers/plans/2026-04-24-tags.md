# Tags — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let any authenticated group member attach short lowercase labels ("tags") to files, then filter the `/` and `/recent` grids by tag and/or uploader via URL query params.

**Architecture:**
- Two new tables: `tags` (unique lowercase name) and `file_tags` (many-to-many join).
- Tags are a flat namespace — no "game:" prefix or namespacing. Users coin tags by attaching a name; creating and reusing tags happens in one API call (upsert by name).
- Filtering is URL-driven (`?tag=<tagId>&uploader=<userId>`) so deep-links and back-button behavior Just Work.
- Uploader filter reuses the existing `uploader_id` column — no new schema for the uploader dimension.
- The existing `listTopLevelItems` helper (from plan #2026-04-24-text-preview-drop-anywhere-folder-roots.md) gains `tagId` and `uploaderId` filter params. When a tag filter is active we drop folders from the mixed grid (tags apply to files only) — the UI still renders folders when no filter is active.

**Tech Stack:** Postgres 16, Next.js 15 App Router, TypeScript strict, Zod for API input, Vitest + testcontainers for DB, jsdom + @testing-library/react for components.

### Task 1: Schema — `tags` and `file_tags`

**Files:**
- Create: `db/init/07-tags.sql`

**Constraints:** Tag name format enforced at the DB level so bad data can't enter through a bypassed API; uniqueness on `name`; join PK is `(file_id, tag_id)`; cascade-delete both directions.

- [ ] **Step 1: Write the migration**

```sql
-- db/init/07-tags.sql
CREATE TABLE IF NOT EXISTS tags (
  id         uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  name       text NOT NULL UNIQUE
               CHECK (name ~ '^[a-z0-9][a-z0-9-]{0,31}$'),
  created_at timestamptz NOT NULL DEFAULT now()
);

CREATE TABLE IF NOT EXISTS file_tags (
  file_id    uuid NOT NULL REFERENCES files(id) ON DELETE CASCADE,
  tag_id     uuid NOT NULL REFERENCES tags(id)  ON DELETE CASCADE,
  created_at timestamptz NOT NULL DEFAULT now(),
  created_by uuid NOT NULL REFERENCES users(id) ON DELETE SET NULL,
  PRIMARY KEY (file_id, tag_id)
);

CREATE INDEX IF NOT EXISTS file_tags_tag_id_idx ON file_tags (tag_id);
CREATE INDEX IF NOT EXISTS file_tags_file_id_idx ON file_tags (file_id);
```

```ts
// app/src/lib/tags.ts
export class TagNameError extends Error {
  constructor(message: string) { super(message); this.name = "TagNameError"; }
}

const TAG_RE = /^[a-z0-9][a-z0-9-]{0,31}$/;

export function normalizeTagName(raw: string): string {
  const lower = raw.trim().toLowerCase();
  if (!lower) throw new TagNameError("tag name is empty");
  if (lower.length > 32) throw new TagNameError("tag name is longer than 32 chars");
  if (!TAG_RE.test(lower)) {
    throw new TagNameError(
      "tag names must be lowercase letters, digits, or hyphens, and can't start with a hyphen",
    );
  }
  return lower;
}
```

```ts
import { pool } from "./db";
// (keep normalizeTagName + TagNameError above)

export type Tag = { id: string; name: string; created_at: Date };
export type TagWithCount = Tag & { file_count: number };

export async function attachTagToFile(
  fileId: string,
  rawName: string,
  actorUserId: string,
): Promise<Tag> {
  const name = normalizeTagName(rawName);
  // Upsert the tag row; RETURNING id,name works on both insert and conflict via DO UPDATE.
  const up = await pool.query<Tag>(
    `INSERT INTO tags (name) VALUES ($1)
     ON CONFLICT (name) DO UPDATE SET name = EXCLUDED.name
     RETURNING id, name, created_at`,
    [name],
  );
  const tag = up.rows[0];
  await pool.query(
    `INSERT INTO file_tags (file_id, tag_id, created_by)
     VALUES ($1, $2, $3) ON CONFLICT DO NOTHING`,
    [fileId, tag.id, actorUserId],
  );
  return tag;
}

export async function detachTagFromFile(fileId: string, rawName: string): Promise<void> {
  const name = normalizeTagName(rawName);
  await pool.query(
    `DELETE FROM file_tags ft USING tags t
     WHERE ft.tag_id = t.id AND ft.file_id = $1 AND t.name = $2`,
    [fileId, name],
  );
}

export async function detachTagFromFileById(fileId: string, tagId: string): Promise<void> {
  await pool.query(
    `DELETE FROM file_tags WHERE file_id = $1 AND tag_id = $2`,
    [fileId, tagId],
  );
}

export async function listTagsForFile(fileId: string): Promise<Tag[]> {
  const { rows } = await pool.query<Tag>(
    `SELECT t.id, t.name, t.created_at
       FROM tags t JOIN file_tags ft ON ft.tag_id = t.id
      WHERE ft.file_id = $1
      ORDER BY t.name ASC`,
    [fileId],
  );
  return rows;
}

export async function listAllTagsWithCounts(): Promise<TagWithCount[]> {
  const { rows } = await pool.query<TagWithCount>(
    `SELECT t.id, t.name, t.created_at,
            COALESCE(
              (SELECT count(*)::int FROM file_tags ft
                 JOIN files f ON f.id = ft.file_id
                WHERE ft.tag_id = t.id AND f.deleted_at IS NULL),
              0
            ) AS file_count
       FROM tags t
       ORDER BY t.name ASC`,
  );
  return rows;
}
```

```ts
// app/src/app/api/files/[id]/tags/route.ts
import { NextResponse } from "next/server";
import { z } from "zod";
import { getCurrentUser } from "@/lib/auth";
import { getFileWithUploader } from "@/lib/files";
import { attachTagToFile, TagNameError } from "@/lib/tags";

const BodySchema = z.object({ name: z.string().min(1).max(64) });

export async function POST(
  req: Request,
  { params }: { params: Promise<{ id: string }> },
) {
  const user = await getCurrentUser();
  if (!user) return NextResponse.json({ error: "unauthorized" }, { status: 401 });
  const { id } = await params;
  const file = await getFileWithUploader(id);
  if (!file) return NextResponse.json({ error: "not found" }, { status: 404 });

  let body: z.infer<typeof BodySchema>;
  try {
    body = BodySchema.parse(await req.json());
  } catch {
    return NextResponse.json({ error: "invalid body" }, { status: 400 });
  }

  try {
    const tag = await attachTagToFile(id, body.name, user.id);
    return NextResponse.json({ tag });
  } catch (err) {
    if (err instanceof TagNameError) {
      return NextResponse.json({ error: "invalid tag name", reason: err.message }, { status: 400 });
    }
    throw err;
  }
}
```

```tsx
// app/src/components/FileTagsEditor.tsx
"use client";
import { useState } from "react";
import { TagChip } from "./TagChip";
import styles from "./FileTagsEditor.module.css";

type Tag = { id: string; name: string };

export function FileTagsEditor({ fileId, initialTags }: { fileId: string; initialTags: Tag[] }) {
  const [tags, setTags] = useState<Tag[]>(initialTags);
  const [draft, setDraft] = useState("");
  const [error, setError] = useState<string | null>(null);
  const [busy, setBusy] = useState(false);

  async function add() {
    const name = draft.trim();
    if (!name) return;
    setBusy(true); setError(null);
    try {
      const res = await fetch(`/api/files/${fileId}/tags`, {
        method: "POST",
        headers: { "content-type": "application/json" },
        body: JSON.stringify({ name }),
      });
      if (!res.ok) {
        const body = await res.json().catch(() => ({}));
        setError(body.reason ?? body.error ?? "couldn't add that tag");
        return;
      }
      const body = await res.json();
      setTags((prev) =>
        prev.some((t) => t.id === body.tag.id) ? prev : [...prev, { id: body.tag.id, name: body.tag.name }]
          .sort((a, b) => a.name.localeCompare(b.name)),
      );
      setDraft("");
    } finally { setBusy(false); }
  }

  async function remove(tag: Tag) {
    const prev = tags;
    setTags((p) => p.filter((t) => t.id !== tag.id));
    const res = await fetch(`/api/files/${fileId}/tags/${tag.id}`, { method: "DELETE" });
    if (!res.ok) setTags(prev); // revert on error
  }

  return (
    <div className={styles.row}>
      {tags.map((t) => <TagChip key={t.id} name={t.name} onRemove={() => remove(t)} />)}
      <div className={styles.addGroup}>
        <input
          className={styles.input}
          placeholder="add tag…"
          value={draft}
          onChange={(e) => setDraft(e.target.value)}
          onKeyDown={(e) => { if (e.key === "Enter") add(); }}
          disabled={busy}
        />
        <button type="button" onClick={add} disabled={busy || !draft.trim()} className={styles.addBtn}>add</button>
      </div>
      {error && <div className={styles.error}>{error}</div>}
    </div>
  );
}
```

```tsx
// app/src/components/FilterBar.tsx
"use client";
import { usePathname, useRouter, useSearchParams } from "next/navigation";
import styles from "./FilterBar.module.css";

type Tag = { id: string; name: string; file_count: number };
type Uploader = { id: string; username: string };

export function FilterBar({ tags, uploaders }: { tags: Tag[]; uploaders: Uploader[] }) {
  const router = useRouter();
  const pathname = usePathname();
  const sp = useSearchParams();
  const tagId = sp.get("tag") ?? "";
  const uploaderId = sp.get("uploader") ?? "";

  function update(next: { tag?: string; uploader?: string }) {
    const params = new URLSearchParams(sp.toString());
    const finalTag = next.tag !== undefined ? next.tag : tagId;
    const finalUploader = next.uploader !== undefined ? next.uploader : uploaderId;
    if (finalTag) params.set("tag", finalTag); else params.delete("tag");
    if (finalUploader) params.set("uploader", finalUploader); else params.delete("uploader");
    const qs = params.toString();
    router.push(qs ? `${pathname}?${qs}` : pathname);
  }

  const active = tagId || uploaderId;

  return (
    <div className={styles.bar}>
      <label className={styles.field}>
        <span className="vv-meta">tag</span>
        <select
          aria-label="filter by tag"
          value={tagId}
          onChange={(e) => update({ tag: e.target.value })}
        >
          <option value="">all</option>
          {tags.map((t) => (
            <option key={t.id} value={t.id}>#{t.name} ({t.file_count})</option>
          ))}
        </select>
      </label>
      <label className={styles.field}>
        <span className="vv-meta">uploader</span>
        <select
          aria-label="filter by uploader"
          value={uploaderId}
          onChange={(e) => update({ uploader: e.target.value })}
        >
          <option value="">anyone</option>
          {uploaders.map((u) => (
            <option key={u.id} value={u.id}>@{u.username}</option>
          ))}
        </select>
      </label>
      {active && <button type="button" onClick={() => update({ tag: "", uploader: "" })} className={styles.clear}>clear</button>}
    </div>
  );
}
```

```ts
export type UploaderRow = { id: string; username: string };
export async function listUploadersWithFiles(): Promise<UploaderRow[]> {
  const { rows } = await pool.query<UploaderRow>(
    `SELECT DISTINCT u.id::text AS id, u.username
       FROM users u JOIN files f ON f.uploader_id = u.id
      WHERE f.deleted_at IS NULL
      ORDER BY u.username ASC`,
  );
  return rows;
```

## Self-Review Notes

- Tag creation is implicit (attach creates if missing). That's simpler than a separate "manage tags" screen and matches user intent ("I want to tag with 'valheim'"). Cleanup of orphan tags (file_count = 0) is deferred — they don't hurt and the user can reuse them. Consider a `GET /api/tags?active=1` filter later if the list grows noisy.
- Tags are case-insensitive and lowercase-normalized on both client and server. The DB CHECK constraint catches any sneaky inserts.
- Filtering by tag drops folders from the grid because folders aren't tagged. Out of scope for this plan: tagging folders. If users request it, the join table needs a polymorphic design (or two tables).
- Tag filter + uploader filter compose (AND), not OR.
- No pagination on the tag-selector dropdown (we expect <100 tags). Revisit if that's wrong.
- Keyboard: `Enter` in the tag input submits; `Esc` behavior not customized (relies on browser defaults).
